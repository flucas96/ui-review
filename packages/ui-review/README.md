# ui-review

Local-first visual feedback for HTML and framework-based web applications. Run `ui-review <url-or-path>` to open an annotated review proxy, or `ui-review mcp --root .` to expose its feedback to an MCP-compatible coding agent. When Claude Code starts a user-scoped MCP server, UI Review automatically uses `CLAUDE_PROJECT_DIR` as the feedback root.

See the repository README for the complete setup, SSH workflow, and Claude Code integration.
